# Домашний магазин

Учебное React-приложение интернет-магазина с каталогом, избранным, корзиной, ленивой загрузкой страниц и тестами.

## Команды

```bash
npm install
npm run dev
npm test
npm run build
```

## Что реализовано

- маршрут `/favourites` со списком избранных товаров;
- удаление товаров из избранного;
- сохранение избранного в `localStorage`;
- состояние через Context API;
- мемоизация через `React.memo`, `useMemo` и `useCallback`;
- ленивая загрузка страниц через `React.lazy` и `Suspense`;
- тесты компонентов, маршрутов, избранного и корзины.
